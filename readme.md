# img-to-ascii

Converts an image to ASCII art.

## Usage

```bash
cargo run <file> <scale>
```

or if you compile it
```bash
cargon build
```
then in the `./target/debug/build/` directory the `img-to-ascii` binary will exist.. throw it in any directory in your $PATH.
